Hotkeys Windows

Ctrl + c

Copy selected text

Ctrl + v

Inset copied text

Ctrl + x

Cut out selected text (Remove from selected place with ability of inserting in another place)

Ctrl + Shift + d

Duplicate string on which cursor is placed or selected part

Ctrl + z

Deny previous action

Needs use

Using hotkeys allows to make repeatable operations or simple by meaning which needs couple actions with mouse or and more clicks(pressing left or right butttons of mouse), so it greatly increase ammount of actions done with keyboard.
